# Snake Social Arena

This is a real-time multiplayer Snake game built with React and Node.js.

## Tech Stack

- Frontend: React 18, Vite, Tailwind CSS
- Backend: Node.js, Express, Socket.IO

## Installation

1. Clone the repository
2. Install dependencies
   ```bash
   cd frontend
   npm install
   cd ../backend
   npm install
   ```

## Usage

### Development

```bash
# Start frontend
cd frontend
npm run dev

# Start backend
cd ../backend
npm run dev
```

### Production

```bash
# Build frontend
cd frontend
npm run build

# Start backend
cd ../backend
npm run start
```
